# News Site

## Overview

This is a simple news site built with React and Tailwind CSS. The site allows users to browse news articles in different categories, such as general, US news, sports, health, tech, politics, entertainment, science, and climate. 
Users can also filter news articles by subcategories (liberal, conservative and independent) under the politics section, so you can get the full story from every angle.

## Features

- Browse news articles in different categories.
- Filter news articles by subcategories under politics.
- Sort news articles by published date.
- Pagination to navigate through the articles.
- Responsive design for mobile and desktop devices.
